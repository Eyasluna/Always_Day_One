### 2. Brew Install
Web: https://docs.brew.sh/Installation
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```
Then:
```
export PATH="/opt/homebrew/bin:$PATH" >> ~/.zshrc
```
